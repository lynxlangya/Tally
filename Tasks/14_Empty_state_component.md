# 14 首页/明细空状态组件

## 目标

- 首页无账单时显示简洁友好的空状态提示。
- 明细页支出/收入排行为空时显示同款空状态。
- 组件化封装，其他页面可复用。
- UI 保持暗色主题、简洁优雅。

## 开发步骤

1. 新建空状态组件
   - 路径：`Core/UIComponents/JOEmptyStateView.swift`
   - 内容：图标 + 主文案 + 可选副文案
   - 支持自定义文案、图标、尺寸与间距

2. 首页接入
   - 当 `HomeViewModel.groups` 为空时，在 List 内显示空状态
   - 视觉上与现有布局对齐（清爽、无边框）

3. 明细排行接入
   - 在 `BillsListRankingView` 中替换“暂无排行”文本为组件
   - 保持滚动区域布局不突兀

## 验收标准

- ✅ 首页无账单时显示空状态提示，且不显突兀
- ✅ 明细页排行为空时显示同款空状态
- ✅ 组件可复用且样式与主题一致
- ✅ 其他页面不受影响
